## Writing Rules

- Final packs contain distilled guidance, not research trails.
- Keep source links, transcripts, and copied excerpts out of published packs.
- Use `templates/reference-pack-template.md` for new packs.
- Update `ai-index.json`, `llms.txt`, and the relevant category README when a pack changes status.
